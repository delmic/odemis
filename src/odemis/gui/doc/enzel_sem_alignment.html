Alignment instructions

Overview
    Alignment procedure to create a coincidental point for the three beams:

  1. Perform the Z stage alignment.
  2. Mill a feature using the FIB for referencing.
  3. Align the E beam relative to the milled feature.
  4. Finish by aligning the FLM relative to the milled feature.
 
SEM-alignment mode
    Perform the SEM alignment via the following steps:

  1. Set the horizontal field width of the SEM in this user interface to a LOW VALUE of approximately XXXXX.
  2. Set the horizontal field width of the FIB on the TFS PC to a LOW VALUE of approximately XXXXX.
  3. Mill a centred feature using the FIB on the TFS PC with an approximated size of XXXX.
  4. Briefly play the FIB view to display the milled feature on the screen
  5. Start playing the SEM view.
  6. Using the beam shift control button centre the feature to the e beam.
  7. Continue to the FLM alignment
