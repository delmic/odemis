Alignment instructions

Overview
    Alignment procedure to create a coincidental point for the three beams:

  1. Perform the Z stage alignment.
  2. Mill a feature using the FIB for referencing.
  3. Align the E beam relative to the milled feature.
  4. Finish by aligning the FLM relative to the milled feature.
 
FLM-alignment mode
    Perform the FLM alignment via the following steps:

  1. Set the horizontal field width of the SEM in this user interface with to a LOW VALUE of approximately XXXXX.
  2. Using both the SEM and FLM view locate where the FLM is currently located on the sample.
  3. Move the objective stage using the controls so the SEM alignment feature is visible.
  4. Align the FLM using the same controls to centre the SEM alignment feature in the FLM.
  5. The alignment is finished, and the current settings are saved.
