Alignment instructions

Overview
    Alignment procedure to create a coincidental point for the three beams:

  1. Perform the Z stage alignment.
  2. Mill a feature using the FIB for referencing.
  3. Align the E beam relative to the milled feature.
  4. Finish by aligning the FLM relative to the milled feature.
